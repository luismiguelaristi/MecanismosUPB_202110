Análisis de posición del mecanismo manivela corredera por métodos numéricos¶

Objetivo¶

Encontrar valores de las variables del mecanismo que me permitan posicionar sus barras, usando mtodos numricos.

Diagramas cinemático y vectorial¶

Usar este enlace para escribir las ecuaciones en formto LaTeX: https://codecogs.com/latex/eqneditor.php

Parmetros: $r2$, $r3$, $\theta_4$

Variable de entrada: $\theta_2$

Variables secundarias: $\theta_3$ y $r_4$

A partir del diagrama vectorial, se puede plantear una ecuacin vectorial que represente al mecanismo:

$\overrightarrow{r_{2}}-\overrightarrow{r_{3}}-\overrightarrow{r_{4}} = 0$ (1)

cuyas ecuaciones escalares son

$r2\cdot cos(\theta_{2})-r3\cdot cos(\theta_{3})-r4\cdot cos(\theta_{4})=0$, (2)

$r2\cdot sin(\theta_{2})-r3\cdot sin(\theta_{3})-r4\cdot sin(\theta_{4})=0$ (3).

El problema ahora consiste en resolver este sistema de ecuaciones NO LINEAL. Para esto usaremos el mdulo de optimizacin de Scipy, especficamente la funcin fsolve.

Solución¶

Primero importamos los mdulos necesarios:

  1. numpy para operaciones numricas
  2. fsolve de scipy.optimize para resolver el sistema de ecuaciones no lineal
In [252]:
import numpy as np
from scipy.optimize import fsolve

Para resolver un sistema de ecuaciones NO LINEAL usando el mtodo numrico de Newton-Rhapson con "fsolve", debemos proveer una FUNCIN.

Repaso sobre funciones¶

Una funcin es un bloque de cdigo reutilizable con entradas y una salida. Se pueden procesar los datos de entrada para generar la salida.

Definamos esta funcin que haga una simple suma:

In [253]:
def funcion(in_1,in_2):
    out = in_1 + in_2
    return out

Si tratamos de ejecutar la funcin, no pasa nada, ya que lo que acabamos de hacer es definir la funcin, pero an no la hemos utilizado.

Ahora para utilizarla:

In [254]:
resultado = funcion(2,2)
print(resultado)

4

Las entradas de la funcin se definen en el parntesis, la salida de la funcin se obtiene al ponerla al lado derecho de la igualdad, en este caso queda almacenada en la variable "resultado".

Lo interesante de las funciones es que son reutilizables.

Usando fsolve¶

Para usar fsolve, debemos incluir el sistema de ecuaciones en una funcin de la siguiente forma:

In [255]:
def manivelaCorredera(condIni, params):
    #variable de entrada y parmetros
    ## Primero la variable de entrada
    th2 = params[0]
    ## Luego los parmetros
    r2 = params[1]
    r3 = params[2]
    th4 = params[3]
    
    # las condiciones iniciales deben darse para que el mtodo funcione
    th3 = condIni[0]
    r4 = condIni[1]
    
    # creamos el vector donde quedar la salida, inicialmente con ceros
    Y = np.zeros(2)
    # Definimos las ecuaciones escalares no lineales
    Y[0] = r2*np.cos(th2) - r3*np.cos(th3) - r4*np.cos(th4);
    Y[1] = r2*np.sin(th2) - r3*np.sin(th3) - r4*np.sin(th4);
    
    return Y

ATENCIÓN¶

Si en este paso obtenemos valores muy diferentes a cero (con valores enteros mayores o menores a cero), debemos revisar varias cosas:

  • las condiciones iniciales estn muy alejadas de la realidad del mecanismo,
  • las ecuaciones escalares estn mal planteadas,
  • los parmetros o la variable de entrada no corresponden con la posicin analizada del mecanismo.

Resolver esto antes de continuar.

for i in range(0,numpos):
    # para cada iteracin actualizamos el valor de la variable de entrada
    parametros[0] = th2V[i]
    # resolvemos con fsolve
    q,info,exitflagV[i],mensaje = fsolve(manivelaCorredera,
                                        x0, 
                                        args = parametros,
                                        full_output = True)
    th3V[i] = q[0]
    r4V[i] = q[1]
    x0 = q
    # Opcional, poner esto para detectar si hay un error
    if exitflagV[i] != 1:
        print('Error en la iteracin ' + str(i))
        print(mensaje)
        break

def animar(i):
